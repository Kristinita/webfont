#### Russian
Подключение несистемных шрифтов с сайта [webfont.ru](http://webfont.ru)

По-умолчанию в плагине будет использоваться английский язык. Для смены языка откройте файл **WebFont.sublime-settings** и строку 
    "lang": "en_US" 
замените на 
    "lang": "ru_RU"
